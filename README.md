# FiveM Strafe Trainer

A simple browser-based training tool for practicing FiveM strafe movement patterns.

## Features

- WASD Pattern Training
- DWAS Pattern Training
- BPM Counter
- Best BPM Tracking
- Streak Counter
- Best Streak Tracking
- Error Counter
- Fullscreen Mode
- Local Storage Save System
- Keyboard Visualizer

## How To Use

1. Hold SHIFT (optional)
2. Perform one of the following patterns:

W → A → S → D

or

D → W → A → S

3. Increase your BPM and consistency.
4. Try to beat your best score.

## Website

https://legittrk.github.io/fivem-strafe-trainer/

## Purpose

This tool is designed to help FiveM players improve movement consistency and build muscle memory for fast strafing.
